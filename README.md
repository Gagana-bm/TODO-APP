# ✅ TaskFlow — Full-Stack To-Do App

A professional full-stack To-Do List application built with React, Supabase, and Tailwind CSS.

## 🔗 Live Demo
👉 [https://todo-app-psi-one-40.vercel.app/](https://todo-app-psi-one-40.vercel.app/)

## 🛠️ Tech Stack
- **Frontend:** React + Vite
- **Styling:** Tailwind CSS
- **Backend & Database:** Supabase
- **Authentication:** Supabase Auth
- **Routing:** React Router
- **Deployment:** Vercel

## ✨ Features
- Email & password authentication (Signup/Login/Logout)
- Add, edit, delete tasks
- Mark tasks as complete/incomplete
- Separate pending and completed task sections
- Each user sees only their own tasks
- Fully responsive design (mobile + desktop)

## 🚀 Getting Started (Local Setup)
1. Clone the repo
```bash
   git clone https://github.com/Gagana-bm/todo-app.git
   cd todo-app
```
2. Install dependencies
```bash
   npm install
```
3. Create a `.env` file in the root folder
